# Kumbara-Kala Internship Project

A modern Android application developed during the VTU Industry & Research Internship at MindMatrix (CL Infotech Pvt. Ltd.), Bengaluru. The project focuses on promoting traditional pottery and artisan services through a digital platform.

## 📌 Project Overview

**Kumbara-Kala** is an Android-based application designed to support local potters and artisans by providing an online platform for showcasing, booking, and managing pottery-related services and products. The application aims to bridge the gap between customers and traditional craftsmen using modern mobile technology.

This internship project was developed using **Kotlin** and **Android Studio** with integrations such as Google Maps, Shared Preferences, and Firebase-based functionalities.

## 🚀 Features

* 👤 User Registration & Login
* 🏺 Pottery Product Display
* 📍 Google Maps Integration
* 📦 Product Booking System
* ❤️ Wishlist / Favorites
* 🔎 Search Functionality
* 📱 Responsive Android UI
* 🔐 Secure Authentication
* 🧾 Order History Management
* ☁️ Cloud Data Storage Support
